## 二、可借鉴功能清单（按优先级排序）

### P0 — 高价值，强烈推荐

#### 1. 电影系列/合集管理
- **参考**：Plex 电影合集、TMDB Collection、TinyMediaManager Movie Sets
- **描述**：将同一系列电影归组（如"哈利波特系列"、"漫威宇宙"），支持折叠展开
- **数据模型**：新增 `Collection` 表（Id, Name, SortOrder, PosterUrl），Movie 增加 `CollectionId` + `CollectionOrder`
- **UI**：列表页增加"合集"视图模式，详情页显示"同系列电影"

#### 2. 高级筛选 & 保存筛选条件 ✅ 已实现
- **参考**：Plex 高级筛选、TinyMediaManager v5 AND/OR 过滤器
- **描述**：多条件组合筛选（年份范围 + 评分范围 + 国家 + 语言 + 片长 + 导演），可保存常用筛选
- **趋势**：TinyMediaManager v5.2 已支持 AND/OR 逻辑组合筛选，成为行业标配
- **UI**：筛选面板增加范围滑块、多选下拉，"保存筛选"按钮

#### 3. 批量编辑
- **参考**：TinyMediaManager 批量编辑、EMDB 批量修改
- **描述**：选中多部电影，批量修改分类、标签、评分、观看状态
- **UI**：列表页多选 + 右侧批量编辑面板

#### 4. 重复电影检测 ✅ 已实现
- **参考**：TinyMediaManager 去重、EMDB 重复检测
- **描述**：检测标题/年份相同的重复电影，提供合并/删除选项
- **实现**：标题+年份匹配算法，支持合并/删除操作
- **UI**：设置页"检测重复"按钮，结果列表 + 操作按钮

### P1 — 中等价值，推荐实现

#### 5. 观影记录 & 日记
- **参考**：Letterboxd 日记功能（核心卖点，2400万用户）
- **描述**：每次观影可写日记（长文本），记录观影感受、影院信息、同行人
- **数据模型**：新增 `WatchLog` 表（Id, MovieId, WatchDate, Content, Location, Companion, Rating）
- **UI**：详情页"观影日记"标签页，时间线展示

#### 6. 电影愿望单
- **参考**：豆瓣想看、Letterboxd Watchlist、Coollector Wishlist
- **描述**：从在线搜索结果直接加入愿望单，不进入主库
- **趋势**：Letterboxd Video Store（2025.12）已实现"发现即观看"体验，愿望单是核心入口
- **数据模型**：新增 `Wishlist` 表或 Movie 增加类型标识
- **UI**：导航栏增加"愿望单"入口

#### 7. 封面管理增强
- **参考**：TinyMediaManager 海报管理、Kodi 图形包
- **描述**：支持手动上传/粘贴封面、从多个在线源选择封面、裁剪调整
- **趋势**：TinyMediaManager v5 已支持多图像类型（海报/背景/Logo/清空图）分别管理
- **UI**：详情页封面区域增加"更换封面"按钮，弹出在线封面选择器

#### 8. 键盘快捷键
- **参考**：Kodi、VLC、EMDB
- **描述**：全局快捷键（Ctrl+F 搜索、Delete 删除、Enter 详情、Esc 返回）
- **实现**：MainWindow KeyBinding + 全局热键注册

#### 9. 影评/短评 & 标签式评价
- **参考**：豆瓣短评、Letterboxd 评价
- **描述**：每部电影可写短评（140字）和长评，支持打标签式评价（如"画面震撼""剧情烧脑"）
- **UI**：详情页增加"我的短评"区域，预设评价标签

### P2 — 锦上添花，可后续迭代

#### 10. 借阅管理
- **参考**：EMDB、MovieCollector、Filmotech
- **描述**：记录电影借给谁了、借出日期、是否归还
- **数据模型**：新增 `Lending` 表（Id, MovieId, Borrower, LendDate, ReturnDate, IsReturned）
- **UI**：详情页"借阅"标签，独立借阅列表视图

#### 11. 数据可视化增强
- **参考**：Letterboxd 年度回顾/年度统计、IMDb Stats
- **描述**：观影热力图（类似 GitHub 贡献图）、导演/演员作品排行、国家分布地图
- **趋势**：Letterboxd Pro 2025 新增年度统计摘要，用户付费意愿强
- **UI**：统计页增加更多图表类型

#### 12. 条码扫描入库
- **参考**：Coollector、My Movies
- **描述**：通过摄像头扫描 DVD/蓝光条码，自动识别并入库
- **实现**：ZXing.Net 条码识别 + 在线数据库查询

#### 13. 插件/扩展系统
- **参考**：Kodi 插件、Plex Channel、MoviePilot 微内核架构
- **描述**：允许第三方开发数据源插件、UI 主题插件
- **趋势**：MoviePilot v2 采用微内核 + 插件热插拔架构，内存占用降低35%
- **实现**：MEF 或自定义插件加载机制

#### 14. AI 智能功能
- **参考**：MoviePilot AI 内容分析、蓝燕云 AI 辅助排期
- **描述**：AI 辅助标签生成、智能影评摘要、观影偏好分析
- **趋势**：2025年38.5%的影视管理软件已集成AI辅助功能，包括自动分类、情感分析等
- **实现**：本地 Ollama/ONNX 模型 或 在线 API（OpenAI/DeepSeek）

#### 15. 跨设备同步
- **参考**：My Movies 多设备同步、Plex 跨平台
- **描述**：支持 WebDAV/云盘同步数据库，多设备共享
- **实现**：SQLite 数据库文件同步 + 冲突合并策略

#### 16. 电视剧/剧集支持
- **参考**：EMDB、TinyMediaManager v5、Plex
- **描述**：扩展支持电视剧管理，按季/集组织
- **趋势**：Letterboxd 计划支持电视剧（2026年仍未上线），市场需求大
- **数据模型**：新增 `TVShow`、`Season`、`Episode` 表

## 三、行业趋势分析（2025-2026）

### 趋势1：AI 赋能成为标配
- 38.5% 的影视管理软件已集成 AI 功能（2025年数据）
- 典型应用：自动标签生成、智能分类、情感分析、观影偏好预测
- MoviePilot 引入 AI 驱动的内容分析引擎，通过视频帧识别自动生成分类标签
- **对 EasyMovie 的启示**：可先从轻量 AI 功能入手（如智能标签推荐），后续考虑本地模型集成

### 趋势2：社交化与社区感
- Letterboxd 用户突破 2400 万，核心是社交+日记+列表
- "发现即观看"体验：Letterboxd Video Store（2025.12）实现从发现到观看的无缝衔接
- 短评/列表/关注/年度回顾成为用户粘性关键
- **对 EasyMovie 的启示**：强化观影日记和短评功能，增加列表/收藏夹管理

### 趋势3：模块化与插件化架构
- MoviePilot v2 采用微内核架构，插件热插拔，内存降低 35%
- TinyMediaManager v5 持续增强过滤器、字幕管理、批量编辑
- 开放 API 生态成为差异化竞争点
- **对 EasyMovie 的启示**：预留插件接口，数据源可扩展

### 趋势4：元数据管理精细化
- TinyMediaManager v5 支持多图像类型（海报/背景/Logo/清空图）分别管理
- IMDB 实验性支持浏览器引擎抓取 + Cookie 认证
- TMDB 支持获取演员/剧组外部 ID（IMDb、TVRage 等）
- **对 EasyMovie 的启示**：增强封面管理，支持多源元数据选择

### 趋势5：跨平台与云同步
- My Movies 支持 Windows/iOS/Android 跨设备同步
- Jellyfin 完全免费开源，Web 端访问
- 云存储原生对接（阿里云盘等）提升传输效率
- **对 EasyMovie 的启示**：保持本地优先定位，可选支持 WebDAV 同步

## 六、EasyMovie 差异化定位

> **本地优先、中文友好、轻量简洁的电影管理工具**

与竞品相比的核心优势：
1. **中文生态**：豆瓣/1905/猫眼多源搜索，拼音检索，中英双语界面
2. **轻量单机**：无需服务器、无需账号、无需网络，SQLite 零部署
3. **简洁易用**：不像 Plex/Kodi 那样重，不像 TinyMM 那样复杂
4. **美观现代**：Material Design + 深色主题 + 海报墙，比同类 WPF 软件好看
5. **数据自主**：本地数据库 + JSON 全量备份，不依赖任何云服务
6. **智能推荐**：基于评分/观看/收藏的本地推荐算法，无需 AI 也能发现好片
7. **性能优先**：批量数据库操作 + 页面预加载 + Visibility 切换，导航零延迟
