## 1. 项目概述

### 1.1 基本信息

| 项 | 内容 |
|---|---|
| **项目名称** | MovieManager |
| **类型** | Windows 桌面应用 |
| **框架** | .NET 9 + WPF |
| **架构模式** | MVVM（CommunityToolkit.Mvvm） |
| **数据库** | SQLite + Entity Framework Core 9 |
| **UI风格** | Material Design（MaterialDesignInXamlToolkit） |
| **图表** | LiveCharts2 |
| **测试** | xUnit + Moq + FluentAssertions |

### 1.2 设计原则

- **先能用，再扩展**：MVP 优先，每个阶段产出可交付的功能
- **数据优先**：好的数据模型是系统的基础，Phase 1 重点打磨模型设计
- **零外部依赖部署**：SQLite 本地数据库，单文件自包含发布
- **豆瓣优先，TMDB 备份**：电影数据源以豆瓣为默认，TMDB 作为降级方案

## 4. 核心数据模型

### 4.1 Movie（电影）

```csharp
public class Movie
{
    public int Id { get; set; }
    public string Title { get; set; }              // 中文片名
    public string OriginalTitle { get; set; }       // 原始片名
    public int Year { get; set; }                   // 上映年份
    public string Director { get; set; }            // 导演
    public string Cast { get; set; }                // 主演（JSON 数组，灵活扩展）
    public string Country { get; set; }             // 国家/地区
    public string Language { get; set; }            // 语言
    public int? Runtime { get; set; }               // 片长（分钟）
    public string Synopsis { get; set; }            // 简介

    // 分类 & 标签
    public int? CategoryId { get; set; }            // FK → Category
    public Category Category { get; set; }
    public ICollection<MovieTag> MovieTags { get; set; }  // 多对多

    // 评分 & 状态
    public int? Rating { get; set; }                // 个人评分 1-10
    public WatchStatus WatchStatus { get; set; }    // 想看/在看/已看
    public DateTime? WatchDate { get; set; }        // 观看日期
    public string Notes { get; set; }               // 简短笔记

    // 封面
    public string CoverImagePath { get; set; }      // 本地封面路径
    public string PosterUrl { get; set; }           // 在线海报 URL
    public string DoubanId { get; set; }            // 豆瓣电影 ID
    public string TmdbId { get; set; }              // TMDB 电影 ID

    // 时间戳
    public DateTime CreatedAt { get; set; }
    public DateTime UpdatedAt { get; set; }
}
```

### 4.2 Category（分类）— 支持层级

```csharp
public class Category
{
    public int Id { get; set; }
    public string Name { get; set; }                // 分类名称
    public string Description { get; set; }         // 描述
    public int? ParentId { get; set; }              // 父分类（自引用）
    public Category Parent { get; set; }
    public ICollection<Category> Children { get; set; }
    public ICollection<Movie> Movies { get; set; }
}
```

### 4.3 Tag（标签）

```csharp
public class Tag
{
    public int Id { get; set; }
    public string Name { get; set; }                // 标签名
    public string Color { get; set; }               // 颜色标记（Hex）
    public ICollection<MovieTag> MovieTags { get; set; }
}
```

### 4.4 MovieTag（多对多关联）

```csharp
public class MovieTag
{
    public int MovieId { get; set; }
    public Movie Movie { get; set; }
    public int TagId { get; set; }
    public Tag Tag { get; set; }
}
```

### 4.5 WatchStatus（枚举）

```csharp
public enum WatchStatus
{
    WantToWatch = 0,   // 想看
    Watching = 1,      // 在看
    Watched = 2        // 已看
}
```

## 6. 技术决策记录

| 决策项 | 选择 | 原因 |
|---|---|---|
| UI 框架 | WPF | 用户主力技术栈，快速交付 |
| MVVM 框架 | CommunityToolkit.Mvvm | 微软官方推荐，轻量 Source Generator |
| 数据库 | SQLite | 零部署依赖，本地单机够用 |
| ORM | EFCore 9 | 官方支持，迁移方便 |
| 图表库 | LiveCharts2 | 开源、WPF 原生支持、API 简洁 |
| UI 组件库 | MaterialDesignInXamlToolkit | Material Design 风格，控件丰富 |
| CSV | CsvHelper | .NET 社区标准 |
| 日志 | Serilog | 结构化日志，文件/控制台双输出 |
| 测试 | xUnit + Moq + FluentAssertions | 三件套，和 CostManagement 一致 |
| 构建 | VS Insiders MSBuild | 避免 dotnet build XAML 编译问题 |

## 7. 风险 & 注意事项

- **豆瓣 API 不稳定**：豆瓣已关闭公开 API，需使用第三方镜像或爬虫方案 → 设计 `IMovieApiClient` 接口，方便随时替换
- **封面图片管理**：本地存 `CoverImagePath`（下载到本地缓存），远程存 `PosterUrl` → 离线也能显示
- **大数据量性能**：卡片视图需启用 UI 虚拟化；列表视图用 DataGrid 内置虚拟化已足够
- **XAML 编译**：必须用 VS MSBuild 构建，`dotnet build` 对 WPF XAML 标记编译支持有问题
